# Python Data Structures & NumPy – Week 2 Internship

This repository contains my **Week 2 internship tasks** focused on Python data structures, string manipulation, and numerical computing using **NumPy**.

The objective of this week was to strengthen my understanding of Python collections and learn how to perform efficient numerical operations using NumPy arrays.

---

## Topics Covered

- Python Data Structures  
  - Lists  
  - Tuples  
  - Sets  
  - Dictionaries  

- String Manipulation  
  - String slicing  
  - Formatting strings  
  - String methods (upper, lower, replace, split, join)

- NumPy Basics  
  - Creating arrays  
  - Understanding array attributes (shape, size, datatype)  
  - Comparing Python lists with NumPy arrays

- Array Operations  
  - Indexing and slicing arrays  
  - Extracting rows and columns  
  - Reshaping arrays  
  - Stacking and concatenation

- Mathematical & Statistical Operations with NumPy  
  - Element-wise operations  
  - Sum, mean, median, min, max  
  - Standard deviation  
  - Aggregation across rows and columns

- Practice Exercises  
  - Matrix operations  
  - Random number generation  
  - Random matrix creation  
  - Simulation examples (e.g., dice rolls)
